## 🎯 Featured Project

### 🚀 **FastTalk** — Real-Time Messaging & Communication Platform
*December 2025*

A production-ready real-time communication platform enabling instant messaging, voice calls, video calls, and screen sharing with enterprise-grade reliability.

#### 🔑 Key Features
- ✨ **Real-Time Messaging** - Instant message delivery using WebSockets with Socket.IO
- 👥 **Online Presence Tracking** - Real-time user status updates
- 📞 **Voice & Video Calls** - WebRTC signaling infrastructure supporting peer-to-peer communication
- 🖥️ **Screen Sharing** - Full-screen sharing capabilities for remote collaboration
- 🔐 **Secure Authentication** - JWT-based auth for REST APIs and persistent WebSocket connections
- 📈 **Scalable Architecture** - Redis-backed Socket.IO adapter for multi-instance deployment
- 🐳 **Production-Ready** - Fully containerized with Docker and Nginx reverse proxy

#### 💼 Technical Stack
**Frontend:** React, TypeScript, Tailwind CSS  
**Backend:** Node.js, Express.js, Socket.IO, WebRTC  
**Database:** MongoDB  
**Cache & Message Broker:** Redis  
**Infrastructure:** Docker, Nginx, Docker Compose  

#### 🏆 Highlights
- Architected scalable real-time system handling concurrent connections
- Implemented secure JWT authentication for both stateless and stateful protocols
- Designed and deployed containerized full-stack application
- Optimized WebSocket adapter for multi-server scalability
